Laundry Management

Put on wings to your Laundry Business

Few Words About The Laundry Management:

This app will manage the laundry and their works, And Put on wings to your Laundry Business.

Work Flow: How The Laundry Management Works
Wash Type List View:
  • Here you can view all available wash types
Create Wash Type:
  • Here you can create new wash type
Other than Wash List View:
  • Here you can view all works other than wash
Create New Work Other than Washing:
  • Here you can create new work other than wash
Clothes Types Kanban view:
Create New Clothe Type:
  • Creating a new clothe type by providing all required fields
  • Here you can specify the specific charges for each laundry work
Wash Orders Kanban View:
  • The orders are grouped by their state
Creating New Laundry Wash Order:
  • Add the lines by selecting the clothe type, wash type etc.
After wash order confirmation:
  • Tracking codes are generated to track the laundry
  • Use the "Print Tracking Code Tags" button to print the tags with barcode
Tracking Tags Printed:
  • Use them to track the laundry
Sent for Wash:
  • When the order is sent for wash, washing works are generated
  • These generated works can be of two types: 1. Washing Work, 2. Make Over Work(Other than Washing)
All Works:
  • Here all the washing works are listed for Laundry order.
  • When the washing work is done for the clothe then other make over works are created.
Wash Work Start:
  • Click the "Start" button to start washing
Laundry Order Status:
  • You can check the updated status for order lines while the order is in process.
Wash Work Finish:
  • Click the "Finish" button to finish washing
Other than Wash work created:
  • If the clothe has works other than wash(make over), they are created when wash is done
All Laundry Work Done:
  • Now click the "Set for Invoice" button to confirm for invoice creation.
Ready for Invoicing:
  • Now click the "Create Invoice" button to create invoice.
Draft Invoice:
  • Now click the "Post" button to set invoice for pay.
Invoice Paid:
Invoice Slip Printed:
Return Clothes:
  • Now click the "Return" button to return the clothes.
Return Clothes Validation Wizard:
  • Here set the quantity out(return) and set the status to return
  • Now click the "Return" button to return the clothes.
Clothes Returned:
